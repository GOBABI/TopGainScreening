## 1. 진입 조건 (모두 충족 시에만 진입)

### 필수 조건
- [ ] 200MA **위**에 있는 종목만 거래
- [ ] 당일 또는 프리마켓 **갭 +3% 이상** (`/pre` 알림 종목)
- [ ] 거래량이 20일 평균 대비 **1.5배 이상** 폭증
- [ ] 주가 **$10 이상** (페니스탁 제외)
- [ ] 시가총액 **$500M ~ $50B** (초소형·대형 제외)

### 기술적 조건
- [ ] RSI **40~75** 사이 (과열·침체 구간 제외)
- [ ] ADX **20 이상** (추세 강도 확인)
- [ ] Qullamaggie 포지션 **b 구간** 우선 (조정 후 재돌파)
- [ ] **거래량 수축** 확인 (베이스 기간 중 거래량 감소 — VCP)
- [ ] `/report` 스코어 **5점 이상** 종목

### 차트 확인 (수동)
- [ ] 주봉 기준 **1~3개월 베이스** 형성 확인
- [ ] 베이스 상단 **돌파 캔들** 확인 (당일 종가 기준)
- [ ] 손절 자리(베이스 하단)까지 **8% 이내**인 종목만

## 2. 진입 금지 조건 (하나라도 해당 시 패스)

- RSI **80 이상** — 단기 과열
- 200MA **아래** — 하락 추세 중
- 갭이 **이미 10% 이상** 벌어진 상태 — 추격 금지
- 공매도 비율 **15% 이상** — 변동성 리스크
- 전체 시장 (`/report` SPY/QQQ) **-1.5% 이상 하락일** — 당일 진입 없음
- VIX **25 이상** — 공포 구간, 포지션 축소
- 베이스 형성 **15일 초과** — 재평가 후 재진입 검토
- 거래량 수축 **미확인** b 구간 — 베이스 품질 의심

## 3. 포지션 사이징

| 조건 | 비중 |
|------|------|
| 스코어 7~9점 + VCP 확인 | 계좌의 최대 **10%** |
| 스코어 5~6점 | 계좌의 최대 **5%** |
| 스코어 4점 이하 | **진입 없음** |
| VIX 20~25 | 비중 **50% 축소** |
| VIX 25 이상 | **진입 없음** |

> 동시 보유 최대 **5종목** — 분산 집중 원칙

## 4. 손절 규칙 (반드시 진입 전 설정)

- 기본 손절: **베이스 하단 -1%** (지지선 이탈 확인)
- 최대 허용 손실: 진입가 대비 **-8%** (절대 초과 금지)
- 갭 하락 시작 당일: **망설임 없이 즉시 손절**
- 손절 후 **당일 재진입 금지** (감정 매매 차단)

## 6. 시장 환경 필터 (`/report` 확인)

| 시장 상태 | 전략 |

## 7. `/pre` → `/report` 연계 워크플로우

```
1. 매일 오전 (한국 오후 10시~) /pre 확인
   → 갭 +3% 이상 종목 리스트 수신

2. 후보 종목 차트 직접 확인 (주봉 베이스)

3. 당일 오전 /report 확인
   → SPY/QQQ/VIX 시장 환경 체크
   → 스코어 5점 이상 + b 구간 + VCP 종목 추출

4. 진입 조건 체크리스트 모두 확인 후 진입

5. 진입 즉시 손절 주문 설정
```

## 8. 심리 규칙

- **FOMO 금지** — 이미 많이 오른 종목 추격 없음
- 손절 후 **복수매매 금지** — 다음 기회 기다림
- 하루 **최대 3번 진입** — 과매매 방지
- 연속 **3번 손절** 시 당일 거래 중단
- 월 수익률 **-10%** 도달 시 2주 휴식
